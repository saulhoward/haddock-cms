NAME

compare-a-file-in-branches.pl

SYNOPSIS

    $ ./compare-a-file-in-branches.pl -d DIR -f FILE

DESCRIPTION

Looks at different versions of a file in different branches of an SVN project and groups them by their MD5 checksum.

Options

-d|--directory

The directory containing the branches.

-f|--file

The file that we want to look at.

This should be relevant to the branch root.

-h|--help

Print this message and exit.

--debug

Run in debug mode.
